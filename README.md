# Pricing-Schemes-for-price-setting-firms
A calculator for pricing schemes as framed by an Industrial Origination 400 level Economics course. Use this to check your answers regarding 1st, second, and 3rd degree pricing calulations.

This project uses real time event handling as a suppliment to exisitng validations.

This project is an example of ways economic majors can create C# applications to more effectively determine the answers to complicated, multi-faceted calculations.
